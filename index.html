The main idea is to train a Neural Network with 80% of the data set provided and try various features to get the best hypothesis, hθ(x); whereby x is the input values. hθ(x) will output the x and y coordinates of the position of your cursor after 0.5 second. Hopefully, the final product will actually predict the position of the cursor relatively well. The Neural Network works by using an initial set of weights(θi(l)), where l corresponds to the layer of activation units or inputs(if first layer) to use to calculate the sum of θ with the activation unit of the corresponding i value. Since, it is supervised Machine Learning, the results(y) can be used with the hypothesis to calculate a cost function J(θ), representing how far the predicted value deviated from the actual result(y). Using an algorithm to minimize J(θ), we can get optimized values of the weights, θ. For a good model, the sum of θi with its corresponding input values(xi) will give an output that is close to the result.

There is a box below this paragraph that records the the coordinates of your cursor every 0.2s in percentage of the width and height of the box. Clicking the start button will start tracking the coordinates of your cursor after 3 seconds.

Data collected will be the initial posiiton of the cursor. Then the distance and angle needed to reach the consecutive position of the cursor. All of which will be calculated and sent to the database automatically by pressing the submit button after recording all 10 points.
